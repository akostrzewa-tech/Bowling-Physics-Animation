# 🎳 Bowling: Physics & Motion Study

## 📖 Overview
This project focuses on **dynamic simulations** and **keyframe animation**. It demonstrates a realistic bowling strike, focusing on proper timing, physics-based object interactions (MassFX), and cinematic camera animation.

## 🎬 Animation Preview
![Bowling Animation Preview](renders/Bowling_Simulation_Preview.gif)
*If the GIF doesn't load, you can find the full video in the `/video` folder.*

## 🛠️ Technical Specifications
* **Software:** 3ds Max 2024
* **Physics Engine:** MassFX (Rigid Body Dynamics)
* **Animation:** Keyframe animation for the ball, procedural physics for the pins.
* **Renderer:** Arnold

## 💡 Key Features
1. **Dynamic Interactions:** Realistic collision detection between the ball and pins.
2. **Motion Timing:** Fine-tuned gravity and friction settings for natural movement.
3. **Camera Work:** Smooth tracking shots to follow the action.
